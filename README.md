# E-Commerce_Recommendation_System
This application is a comprehensive web platform designed to improve the online shopping experience through machine learning. It provides a direct bridge between user interest and product discovery by analyzing item attributes to offer relevant suggestions tailored to specific searches.

The core of the system includes a dynamic homepage that highlights trending products to engage users immediately upon arrival. A central search feature allows visitors to find specific items and instantly view a list of similar products generated by a content-based recommendation engine. Beyond product discovery, the platform also manages secure user account functions such as registration and login to maintain a consistent browsing session.

The backend is powered by Python and the Flask web framework while data management is handled through Pandas for product processing and SQLAlchemy for database interaction. The recommendation logic utilizes the Scikit-Learn library to perform TF-IDF vectorization and calculate Cosine Similarity between product tags. This technical stack ensures that the application is both efficient and capable of handling complex mathematical comparisons.

When a user interacts with the search bar, the system converts text-based tags into numerical vectors to measure how closely related different items are. This mathematical comparison allows the application to rank and display the most relevant products within a clean interface built on Bootstrap. The data is stored and retrieved using a MySQL database named ecom which handles all user-specific information and product metadata.

The project requires a standard Python environment with libraries for web development and data science. A local MySQL server is also necessary to support the authentication features and product data storage. This system serves as a functional prototype for modern e-commerce architectures by combining traditional web development with automated data analysis.
